Лаба 2.
Настроить конфигурирование созданных Bean из первой лабы в соответсвии с приведенными способами.

Например:
1) Для первого Bean настроить его аргументы конструктора.

2) Для второго Bean настроить его аргументы конструктора и сеттеры, которые передают дополнительные аргументы не описанные в конструкторе.

3) Для третьего Bean настроить конструктор по умолчанию, который задает значения по умолчанию, однако передать все свойсва через сеттеры.

4) Так-же определить, какие bean должны работать по singleTon а какие Prototype.

5) В соответсвии с этим определить методы инициализации и по необходимости методы деструкторы.
